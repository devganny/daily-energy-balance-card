# Changelog

Alle wichtigen Änderungen an diesem Projekt werden in dieser Datei dokumentiert.

Das Format basiert auf [Keep a Changelog](https://keepachangelog.com/de/1.0.0/),
und dieses Projekt folgt [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-XX

### Hinzugefügt
- Erste Version der Daily Energy Balance Card
- Grafische Darstellung von PV-Leistung, Netzbezug/-einspeisung und Batterie
- Unterstützung für Dark/Light Mode
- Konfigurierbare Sensoren für alle Energiequellen
- Responsive Design
- HACS-Integration
- Vollständige Dokumentation

### Features
- Echtzeit-Datenvisualisierung
- Automatische Skalierung der Balken
- Farbkodierte Darstellung der verschiedenen Energiequellen
- Unterstützung für Auto-Verbrauch (E-Mobilität)
- Konfigurierbare Kartengröße
- Fehlerbehandlung und Ladeanimationen

### Technische Details
- JavaScript-basierte Implementierung
- SVG-Rendering für scharfe Darstellung
- Shadow DOM für CSS-Isolation
- HomeAssistant Lovelace Card API
- HACS-kompatible Struktur
